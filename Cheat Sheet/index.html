HTML (HyperText Markup Language) is the standard language for creating web pages and web applications. It describes the structure of a webpage using markup. HTML elements are represented by tags, which label pieces of content such as headings, paragraphs, links, images, and other items.

Flexbox

Flexbox (Flexible Box Layout) is a CSS layout model that makes it easy to design flexible and efficient layouts, align items, and distribute space within a container, even when their size is unknown or dynamic. It is especially useful for building one-dimensional layouts (rows or columns) and for responsive design.

Step-by-Step Guide to Using Flexbox

  1. Create a container element in your HTML file:
    <div class="container">...</div>
  2. Set display: flex; in your CSS file on the container:
    .container { display: flex; }
  3. Add child elements (flex items): in your HTML file
    <div class="item">1</div> <div class="item">2</div> ...
  4. Control direction with flex-direction in your CSS file:
    .container { flex-direction: row; } (default is row, can also be column)
  5. Allow wrapping with flex-wrap in your CSS file:
    .container { flex-wrap: wrap; }
  6. Align items with justify-content and align-items in your CSS file:
    .container { justify-content: center; align-items: center; }
  7. Control spacing with gap in your CSS file:
    .container { gap: 1em; }
  8. Use flex on items for flexible sizing: in your CSS file
    .item { flex: 1; }
  9. Override alignment for a single item with align-self in your CSS file:
    .item.special { align-self: flex-end; }

Example Code 1

Don't forget to link your files to each other in the Head section of your HTML file


<div class="container">
  <div class="item">1</div>
  <div class="item special">2</div>
  <div class="item">3</div>
</div>
            

.container {

  flex-wrap: wrap;
  justify-content: center;
  align-items: center;
  gap: 1em;
}
.item {
  flex: 1;
}
.item.special {
  align-self: flex-end;
}

CSS Grid

CSS Grid Layout is a powerful 2-dimensional layout system for the web. It allows you to create complex layouts easily by defining rows and columns, and placing items precisely within a grid. Grid is ideal for building both simple and advanced layouts, and works well alongside Flexbox.

Step-by-Step Guide to Using Grid

  1. Create a grid container in your HTML file:
    <div class="grid-container">...</div>
  2. Set display: grid; in your CSS file on the container:
    .grid-container { display: grid; }
  3. Define columns and rows with grid-template-columns and grid-template-rows:
    .grid-container { grid-template-columns: 1fr 1fr 1fr; grid-template-rows: auto; }
  4. Add grid items as children:
    <div class="grid-item">1</div> ...
  5. Optionally, place items using grid-column and grid-row:
    .grid-item.special { grid-column: 2 / 4; grid-row: 1; } 

<div class="grid-container">
  <div class="grid-item">1</div>
  <div class="grid-item special">2</div>
  <div class="token attr-name">grid-item">3</div>
</div>
            

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 1em;
  background: #23272e;
  padding: 1em;

  background: #4FC3F7;

  font-weight: bold;
  border-radius: 6px;
  display: flex;
  align-items: center;
  justify-content: center;
  min-height: 60px;
}
.grid-item.special {
  grid-column: 2 / 4;
  grid-row: 1;
  background: #ffb86c;
  color: #23272e;
}
